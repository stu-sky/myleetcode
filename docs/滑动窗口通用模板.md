# 滑动窗口通用模板

## 什么时候用滑动窗口？

遇到以下关键词，优先考虑滑动窗口：

- **连续**子串 / 子数组
- **最长** / **最短** / **恰好** 满足某条件
- 条件涉及：不重复、包含特定字符、和 ≥ 某值等

## 核心模板（可变窗口）

```go
left := 0
// 维护窗口状态的数据结构（map、计数器等）

for right := 0; right < len(s); right++ {
    // 第一步：将 s[right] 加入窗口（更新状态）

    for 窗口不满足条件 {
        // 第二步：将 s[left] 移出窗口（更新状态）
        left++
    }

    // 第三步：更新答案
}
```

**三步永远不变**：加入 → 收缩 → 更新。

## 两类题的区别

| 类型 | 求什么 | 更新答案的位置 | 收缩条件 |
|------|--------|---------------|----------|
| **求最长** | 最长满足条件的窗口 | 收缩**之后**（窗口合法时取 max） | 窗口不合法时收缩 |
| **求最短** | 最短满足条件的窗口 | 收缩**之前/中**（窗口合法时取 min） | 窗口合法时继续收缩 |

### 求最长模板

```go
for right := 0; right < len(s); right++ {
    // 加入 s[right]

    for 窗口不合法 {
        // 移出 s[left]
        left++
    }

    ans = max(ans, right-left+1) // 窗口合法，取最长
}
```

### 求最短模板

```go
for right := 0; right < len(s); right++ {
    // 加入 s[right]

    for 窗口合法 {
        ans = min(ans, right-left+1) // 窗口合法，取最短
        // 移出 s[left]
        left++
    }
}
```

注意区别：求最长时，**不合法才收缩**；求最短时，**合法就收缩**（试图找更短的）。

## 用模板套四道经典题

### 1. 无重复字符最长子串（第 3 题）

- **求**：最长
- **窗口状态**：`map[byte]int` 记每个字符出现次数
- **不合法条件**：`window[c] > 1`（有重复）
- **收缩动作**：`window[s[left]]--; left++`

```go
for right := 0; right < len(s); right++ {
    window[s[right]]++
    for window[s[right]] > 1 {
        window[s[left]]--
        left++
    }
    ans = max(ans, right-left+1)
}
```

### 2. 长度最小的子数组（第 209 题）

- **求**：最短
- **窗口状态**：`sum` 记窗口内元素和
- **合法条件**：`sum >= target`
- **收缩动作**：`sum -= nums[left]; left++`

```go
for right := 0; right < len(nums); right++ {
    sum += nums[right]
    for sum >= target {
        ans = min(ans, right-left+1)
        sum -= nums[left]
        left++
    }
}
```

### 3. 找所有字母异位词（第 438 题）

- **求**：所有窗口大小 = len(p) 且字符频次 = p 频次的起始下标
- **窗口状态**：`[26]int` 记窗口内字符频次 + `valid` 计数
- **合法条件**：`valid == 目标字符种类数` 且 `right-left+1 == len(p)`

```go
for right := 0; right < len(s); right++ {
    // 加入 s[right]，更新 valid
    for right-left+1 > len(p) {
        // 移出 s[left]，更新 valid
        left++
    }
    if valid == need的种类数 {
        result = append(result, left)
    }
}
```

### 4. 最小覆盖子串（第 76 题）

- **求**：最短
- **窗口状态**：`map[byte]int` 记窗口内字符频次 + `valid` 计数
- **合法条件**：`valid == 目标字符种类数`（窗口已包含目标所有字符）
- **收缩动作**：更新答案后，移出 `s[left]`，更新 valid

```go
for right := 0; right < len(s); right++ {
    // 加入 s[right]，若某字符达标则 valid++
    for valid == need的种类数 {
        // 窗口合法，取最短
        ans = min(ans, right-left+1)
        // 移出 s[left]，若某字符不达标则 valid--
        left++
    }
}
```

## 模板使用 checklist

遇到新题时，按以下步骤套模板：

1. **判断求最长还是最短**（决定收缩条件和更新位置）
2. **确定窗口状态用什么维护**（map、数组、计数器、sum 等）
3. **写出不合法/合法条件**（for 循环的判断）
4. **写出加入和移出的更新逻辑**（通常是对称的：加入时 ++，移出时 --）
5. 套进模板，结束

> **核心原则**：所有可变滑动窗口题都是「加入 → 收缩 → 更新」三步。变的只是窗口状态和收缩条件，模板本身不变。
