# 3. 无重复字符的最长子串

**代码**：[codes/0003-longest-substring-without-repeating-characters.go](../codes/0003-longest-substring-without-repeating-characters.go)

## 题目

给定一个字符串 `s`，找出其中不含重复字符的**最长子串**的长度。

**示例**：
- 输入：`s = "abcabcbb"` → 输出：`3`（"abc"）
- 输入：`s = "bbbbb"` → 输出：`1`（"b"）
- 输入：`s = "pwwkew"` → 输出：`3`（"wke"）

## 思路

### 问题本质

找**最长**的连续子串，使得窗口内**无重复字符**。  
→ 典型的滑动窗口题：right 扩展窗口，当窗口不满足条件时 left 收缩。

### 解法：标准滑动窗口

用 `map[byte]int` 记录窗口内每个字符的出现次数。

1. `right` 右移，把 `s[right]` 加入窗口
2. 如果 `s[right]` 在窗口内出现 > 1 次（有重复），不断右移 `left` 并移出字符，直到没有重复
3. 更新答案 `ans = max(ans, right-left+1)`

### 示例

```
s = "abcabcbb"

right=0('a'): window={a:1}, 无重复, ans=1
right=1('b'): window={a:1,b:1}, 无重复, ans=2
right=2('c'): window={a:1,b:1,c:1}, 无重复, ans=3
right=3('a'): window={a:2,...}, a重复 → 收缩left
  left=0: 移出'a', window={a:1,b:1,c:1}, left=1, 不再重复
  ans=3 (right-left+1 = 3-1+1 = 3)
right=4('b'): window={...,b:2}, b重复 → 收缩left
  left=1: 移出'b', window={a:1,c:1,b:1}, left=2, 不再重复
  ans=3
...最终 ans=3
```

### 复杂度

- 时间 O(n)：left 和 right 各最多走 n 步
- 空间 O(字符集大小)：最多 128 个 ASCII 字符

## 易错点

1. **收缩条件**：是 `window[c] > 1`（当前字符重复了），不是 `len(window) > 某值`。
2. **更新答案的位置**：在收缩**之后**更新，此时窗口一定合法。
3. **left 不能跳跃**：标准模板里 left 逐步移动（每步减去移出字符的计数），保证 window 状态始终正确。

## 你的写法 vs 标准模板

| 对比 | 你的写法（跳跃式） | 标准模板（逐步收缩） |
|------|-------------------|---------------------|
| start 移动 | 直接跳到 `value+1` | 逐步 `left++` |
| 性能 | 略优（跳过中间） | 同为 O(n) |
| 通用性 | 仅适合「无重复」这一道 | 适用所有滑动窗口题 |
| 记忆成本 | 需要单独记这种跳跃逻辑 | 一套模板通吃 |

**建议**：用标准模板，一套模板解决 3、76、438、209 等所有滑动窗口题。

## 变种思路

| 变种 | 思路 |
|------|------|
| **至多 k 个不同字符（340）** | 收缩条件改为 `len(window) > k` |
| **找所有异位词（438）** | 固定窗口大小 = 目标串长度 |
| **最小覆盖子串（76）** | 收缩条件改为「窗口已包含目标所有字符」，取最短 |
