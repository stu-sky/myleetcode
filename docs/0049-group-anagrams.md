# 49. 字母异位词分组

**代码**：[codes/0049-group-anagrams.go](../codes/0049-group-anagrams.go)

## 题目

给你一个字符串数组，请你将**字母异位词**组合在一起。可以按任意顺序返回结果列表。

字母异位词是由重新排列源单词的字母得到的一个新单词，所有源单词中的字母通常恰好只用一次。

**示例**：
- 输入：`strs = ["eat","tea","tan","ate","nat","bat"]`
- 输出：`[["bat"],["nat","tan"],["ate","eat","tea"]]`

## 思路

### 问题本质

字母异位词的定义：**字符组成完全相同，只是顺序不同**。  
→ 换句话说：每个字母的出现次数相同。  
→ 因此，只要找到一种「与顺序无关」的特征，就能把异位词归到同一组。

### 解法：字符计数

用每个字符串的**字符频次**作为分组 key：统计 a~z 各出现几次，得到 `[26]int`，相同频次的归为一组。  
Go 中 `[26]int` 定长数组可直接作 map key，无需转字符串。  
时间复杂度 O(n·k)，空间 O(n·k)。

### 示例

```
"eat" → count: a=1, e=1, t=1 → key₁
"tea" → count: a=1, e=1, t=1 → key₁  （与 eat 相同）
"ate" → count: a=1, e=1, t=1 → key₁
"tan" → count: a=1, n=1, t=1 → key₂

分组结果：[["eat","tea","ate"], ["tan"]]
```

## 易错点

1. **key 类型**：`[]int` 切片不能作 map key（Go 中切片不可比较），需用 `[26]int` 定长数组。
2. **字符范围**：题目限定小写字母，用 `c-'a'` 得到 0~25 下标；若有其他字符需先过滤或换结构。
3. **遍历顺序**：`range map` 顺序随机，但题目允许任意顺序返回，无需额外处理。

## 变种思路

| 变种 | 思路 |
|------|------|
| **仅判断两串是否异位**（如 242） | 两个 `[26]int` 逐位比较 |
| **找异位词索引** | 同上分组，返回每组在原数组中的下标 |
| **含 Unicode** | 用 `map[rune]int` 替代 `[26]int` |
